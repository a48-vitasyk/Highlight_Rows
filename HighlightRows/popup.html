Збіги (усі сторінки)

🏷 тег · 🔒 блок · ⏰ будильник
Тікети з усіх сторінок черги, що підпадають під теги / блокування / будильники. Клік — відкрити тікет.

Без відповіді

Тікети, де від останньої відповіді підтримки минуло більше порога. Оновлюється ~кожні 15 хв, поки відкрита панель.

Заблокований запит

0 - нагадати лише один раз.

Правила за тегами

Збіг - якщо тема тікета містить тег. 🔔 - сповіщення, 🔊 - звук, остання комірка - повтор (хв, 0 = один раз).

Трафік

При відкритті тікета вписує «Трафік X.XX / Y.YY TB» у блок «Информация о запросе».

Будильники

ID тікета + час. Спрацьовує за 5 хв до часу. 🔔 активний / 🔕 заглушено (клік перемикає на сьогодні), × - видалити.
